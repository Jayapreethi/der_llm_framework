# DER LLM Framework

A secure containerized framework for multi-cloud platforms

## Overview

GPT-powered ML Data Analysis Service for Distributed Energy Resources (DER) with comprehensive analytics and monitoring capabilities.

## Quick Start

Navigate to the demo directory and run the setup:

```powershell
cd der_llm_framework/demo
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force
.\setup.ps1
```

## Features

- **Data Service**: DER data aggregation and processing
- **LLM Service**: GPT-powered analysis and insights
- **ML Analytics**: Clustering, anomaly detection, predictive analysis
- **Metrics Dashboard**: Real-time system and performance monitoring
- **Docker Containerized**: Easy deployment and scaling

## Documentation

See `demo/README.md` for detailed setup and usage instructions.

## Scripts

- `setup.ps1` - Full environment setup and service deployment
- `show_metrics.ps1` - Performance and system metrics snapshot
- `dashboard.ps1` - Interactive ASCII dashboard
- `test_table_output.ps1` - Endpoint testing and validation

## Requirements

- Docker Desktop
- PowerShell (Windows)
- OpenAI API Key (optional, for GPT features)
